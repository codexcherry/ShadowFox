## 🚀 Get Started in Minutes

1. **Clone**:  
   <kbd>git clone https://github.com/your-repo/GreenIntel.git</kbd>

2. **Install**:  
   <kbd>pip install -r requirements.txt</kbd>

3. **Configure**:  
   <kbd>
   NASA_API_KEY=your_api_key<br>
   HG_FACE_API=your_api_key
   </kbd>

4. **Launch**:  
   <kbd>python run.py</kbd>  
   or  
   <kbd>streamlit run app/main.py</kbd>

## 🧠 <span style="color:#ffd600;">AI & NLP Highlights</span>

- **FLAN-T5-Small** — Chosen for its agile, instruction-following power and efficient deployment in real-time greenhouse systems.
- **Rule-based fallbacks** for reliability when APIs are down.
- **Domain knowledge**: Crop, climate, and greenhouse expertise integrated.
- **RAG (Retrieval-Augmented Generation)**: Contextual Q&A using real-time environmental logs.
- **Research-first methodology**:  
  <ul>
    <li>Domain expertise benchmarking</li>
    <li>Contextual understanding with and without RAG</li>
    <li>Reliability & response quality analysis</li>
    <li>Ethical: Bias, transparency, and impact checks</li>
  </ul>
- **Visual, animated performance metrics** — Accuracy, error, historical trends, and crop scoring.

## 🛠️ <span style="color:#ffd600;">How It Works</span>

1. **Choose Your Location**: Pick any region on the globe with the map.
2. **Get NASA Data**: Instantly pull weather and soil data for your greenhouse.
3. **See Smart Suggestions**: AI agents analyze conditions, report trends, and recommend actions.
4. **Ask Anything**: The LLM assistant answers your greenhouse questions in real time—context-aware, reliable, and clear.
5. **Visualize Everything**: Animated graphs and gauges make understanding easy and beautiful.
